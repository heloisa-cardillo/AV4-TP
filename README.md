# Atlantis

Interface web SPA (Single Page Application) do sistema Atlantis, desenvolvida com React + TypeScript + Vite. Esta é a AV4 da disciplina de Tecnologia de Programação da FATEC São José dos Campos, baseada no sistema CLI desenvolvido na AV3.

## Como rodar

**Pré-requisitos**
- Node.js instalado
- npm instalado

**Instalação e execução**

```bash
# Instalar dependências
npm install

# Rodar em modo desenvolvimento
npm run dev
```

O sistema estará disponível em `http://localhost:5173`.

## Tecnologias utilizadas

- React 19
- TypeScript
- Vite
- localStorage
